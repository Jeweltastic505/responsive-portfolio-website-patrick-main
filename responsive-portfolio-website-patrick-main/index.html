Java - Pirate Themed Game

Project Description:

The primary focus of this project was to build a Java Game using the City Engine based on a selected theme.

My Tasks:

I developed an interactive pirate-themed game using Java, IntelliJ, and the City Server. This captivating game spans four unique levels, each symbolizing the elemental forces of fire, water, air, and earth. Within these levels, players encounter a diverse range of challenges, featuring unique objects, power-ups, enemy characters and obstacles that demand navigation within specified time limits.

In elevating the gaming experience, I've integrated a user-friendly Graphical User Interface (GUI) toolbar and a login page. This intuitive toolbar not only streamlines the level selection process but also empowers players to fine-tune sound settings and access an instruction manual for guidance. Additionally, upon launching the game, the login page enables users to personalize their experience by adding their names and facilitates progress saving, contributing to a more personalized and engaging gameplay. Furthermore, I have carefully selected music and sound effects for the theme of each level to enhance the overall ambiance, creating a polished and professional finish to the gaming experience.
